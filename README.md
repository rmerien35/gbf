This is a Grammar compiler in Java for studies purposes.

gram_gbf.dat is the syntax of type "Grammaires Bien Formees" (GBF) (Well-Formed Grammars in english)

Use the GBF program to analyse your grammar : is it well formed ?
Use the Syntax program to compile your grammar.
